# 🏥 MediCare – Pharmacy Management System

🚀 After multiple attempts, restarts, and self-doubt… I finally built this!

This project was started around **December 2025** and completed after **4 months of consistent effort**.  
It was one of the most challenging yet rewarding projects in my development journey.

---

## 💡 Project Overview

**MediCare** is a **full-stack web application** designed to manage pharmacy operations efficiently with separate **Admin and User panels**.

---

## 🚚 Order System

There are 3 types of delivery systems in real-world applications:

1. Home Delivery (like Flipkart)  
2. Store Pickup  
3. Hybrid (Both options)

👉 This project implements **Store Pickup System**  
Users place orders online and visit the store to collect medicines.

---

## 👤 User Features

- View medicines on homepage with categories
- Browse categories (Tablets, Syrups, Baby Care, etc.)
- User Authentication (Login / Signup)

### 🛒 Order System:
- Redirect to login if user is not authenticated
- Orders stored in database
- Orders visible in Admin Dashboard

### ➕ Additional Features:
- Book doctor appointments
- View doctors added by admin
- Contact form (via PHPMailer SMTP)
- Logout functionality
